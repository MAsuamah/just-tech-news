# Just Tech News
## Description 
A tech news website where users can post, upvote, and comment on links to news articles.

## How to Use
Create an account so can post, upvote, and comment.

## Built With
* Sequelize 
* MySQL2
* Node.js
* Express.js
* Handlebars.js
